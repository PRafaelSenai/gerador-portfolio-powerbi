Como Publicar Online
🐙 PARTE 1 — GitHub Pages (link gratuito)
1

Criar conta

Acesse github.com → Sign Up → conta gratuita com seu e-mail.

2

Criar repositório especial

Clique em New → no nome coloque exatamente: seuusuario.github.io → marque Public → Create repository.

3

Fazer upload do index.html

Add file → Upload files → arraste o arquivo index.html que você baixou → escreva "Portfólio" na mensagem → Commit changes.

4

Acessar o site

Aguarde ~2 minutos e acesse: https://seuusuario.github.io 🎉

💼 PARTE 2 — LinkedIn
1

Adicionar ao perfil

Editar perfil → Websites → cole https://seuusuario.github.io com rótulo "Portfólio Power BI".

2

Fixar em Destaque

Perfil → Adicionar seção → Em Destaque → Link → cole o link. Fica fixado no topo do perfil.

3

Publicar post
