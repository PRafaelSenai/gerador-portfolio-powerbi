Como Publicar Online — Passo a Passo
🐙 PARTE 1 — Publicar no GitHub Pages (Site Gratuito)
1

Criar conta no GitHub

Acesse github.com e crie uma conta gratuita com seu e-mail.

2

Criar repositório especial

Clique em New → Nome: seuusuario.github.io → Marque Public → Create repository.

3

Fazer upload do index.html

Add file → Upload files → Arraste o index.html → escreva "Portfólio inicial" → Commit changes.

4

Acessar seu site

Em até 2 minutos estará online em: https://seuusuario.github.io 🎉

5

Atualizar no futuro

Delete o arquivo antigo e faça upload do novo index.html. O site atualiza automaticamente.

💼 PARTE 2 — Compartilhar no LinkedIn
1

Adicionar o link no perfil

Editar perfil → Websites → adicione https://seuusuario.github.io com rótulo "Portfólio Power BI".

2

Fixar em "Em Destaque"

Adicionar seção → Em Destaque → Link → cole o link. Fica no topo do perfil — primeira coisa que o recrutador vê.

3

Publicar post de lançamento

Use o modelo abaixo. Adicione um print do portfólio ao post — posts com imagem têm 3x mais alcance.
